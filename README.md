## üì¶ Core NPM Packages

This project leverages a robust set of libraries for its frontend and backend functionalities:

| `firebase` | Provides backend services, including authentication and potentially database (Firestore/Realtime DB).
| `react` & `react-dom` | The fundamental libraries for building the component-based user interface.
